XPath String functions example

Posted by: Chandan Singh in XPath March 3rd, 2015

In the previous examples, we talked about how we can query for a particular node(s) and extract the attribute value from a node in an XML Document.

In this example, we shall see what other String operations on XPath are supported by the Java programming language.

The String operations are a range of functions that may be used to search/query. These functions operate on string variables or return a string in specific format.

 
 

Supported String functions:

Java supports the following String functions on XPath:

  • text
  • concat
  • starts-with
  • contains
  • substring-before
  • substring-after
  • substring
  • string-length
  • normalize-space
  • translate

Apart from these, there are several over-loaded versions of the methods mentioned above. We shall discuss all those methods in detail.

We will use the following XML file for our further examples.

Examples:

1. text()

The text() method is used for the string representation of the node currently selected.

view sourceprint?
1XPathExpression expr = xpath.compile("//cricketer[@type='righty']/name/text()");
2String cricketer = (String) expr.evaluate(doc, XPathConstants.STRING);
3System.out.println("The righty cricketer is : " + cricketer);

Output:

view sourceprint?
1The righty cricketer is : Shikhar Dhawan

2. concat(String arg1,String arg2,String… arg3)

The concat(String arg1,String arg2,String... arg3) method is used to concat strings from the evaluation of two or more XPath expressions.

view sourceprint?
1XPathExpression expr = xpath.compile("concat(//cricketer[name='Shami']/@type,//cricketer[name='Zaheer Khan']/@type)");
2String combination = (String) expr.evaluate(doc, XPathConstants.STRING);
3System.out.println("The concat result is : " + combination);

Output:

view sourceprint?
1The concat result is : rightylefty

The arguments for concat method can also be static string objects.

view sourceprint?
1XPathExpression expr = xpath.compile("concat(//cricketer[name='Shami']/@type,' Bowler'");

Output:

view sourceprint?
1The concat result is : righty Bowler

3. contains(String target, String tosearch)

This method is used to search for a particular string in a target string. If the need is to find nodes with a particular String pattern, we use contains method. Look at the below code snippet how it is used.

view sourceprint?
1XPathExpression expr = xpath.compile("//cricketer[contains(name,'MS')]/name/text()");
2String msdesc = (String) expr.evaluate(doc, XPathConstants.STRING);
3System.out.println("The Player with name containing 'MS' is : " + msdesc);

Output:

view sourceprint?
1The Player with name containing 'MS' is : MS Dhoni

4. starts-with(string1, string2):

As the name indicates, the starts-with method determines if the a particular tag in the node starts with a particular string. Look at the example below to see how it is used:

view sourceprint?
1XPathExpression expr = xpath.compile("//cricketer[starts-with(name,'Za')]/name/text()");
2String startswith = (String) expr.evaluate(doc, XPathConstants.STRING);
3System.out.println("The Player name starting-with 'Za'  is : " + startswith);

Output:

view sourceprint?
1The Player name starting-with 'Za'  is : Zaheer Khan

5. substring-before(String target, String tosearch)

The substring-before is used to extract the part of string from the beginning of the string to the position where the second string in the argument starts. The example below demonstrates this clearly:

view sourceprint?
1XPathExpression expr = xpath.compile("substring-before(//cricketer[name='MS Dhoni']/position,'-Keeper')");
2String substrbefore = (String) expr.evaluate(doc, XPathConstants.STRING);
3System.out.println("The substring before Keeper is : " + substrbefore);

Output:

view sourceprint?
1The substring before Keeper is : Wicket

6. substring-after(String target, String tosearch)

The substring-before is used to extract the part of string from the first occurrence of the second string argument to the end of the string. The example below demonstrates this clearly:

view sourceprint?
1XPathExpression expr = xpath.compile("substring-after(//cricketer[name='MS Dhoni']/position,'Wicket-')");
2String substrafter = (String) expr.evaluate(doc, XPathConstants.STRING);
3System.out.println("The substring after Keeper is : " + substrafter);

Output:

view sourceprint?
1The substring before Keeper is : Keeper

7. substring(String target, int startingindex, int length)

The XPath offers a number of overloaded substring functions to work. The substring(String target, int startingindex, int length) method is used to extract a sub-string of specified length from the index position specified. There are other substring functions that maybe used as per the requirement. Lets look at an example of the substring :

view sourceprint?
1XPathExpression expr = xpath.compile("substring(//cricketer[name='MS Dhoni']/position,'1','4')");
2String substr = (String) expr.evaluate(doc, XPathConstants.STRING);
3System.out.println("The substring is : " + substr);

Output:

view sourceprint?
1The substring is : Wick

8. string-length(string)

This method is used to calculate the size of the string. Another overloaded version is the string-length(). This method is used to get the size of the currently selected node.

view sourceprint?
1XPathExpression expr = xpath.compile("string-length(//cricketer[name='MS Dhoni']/position)");
2String strlength = (String) expr.evaluate(doc, XPathConstants.STRING);
3System.out.println("The length of the string is : " + strlength);

Output:

view sourceprint?
1The length of the string is : 13

9. normalize-space(string)

This method is used to remove the leading and trailing spaces of the string. Another overloaded version is the normalize-space(). This one is used to remove the leading and trailing spaces of the currently selected node.

view sourceprint?
1XPathExpression expr = xpath.compile("normalize-space(//cricketer[name='Shikhar Dhawan']/role/text())");
2String result = (String) expr.evaluate(doc, XPathConstants.STRING);
3System.out.println("The normalized string is : " + result);

Output:

view sourceprint?
1The normalized string is : Batsman

10. translate(String targetString, String stringtoreplace, String replacementstring)

The translate method is used to replace the occurrences of a particular string with another string. An example will show how:

2String replacedstring = (String) expr.evaluate(doc, XPathConstants.STRING);
3System.out.println("The translated string is : " + replacedstring);

Output:

view sourceprint?
