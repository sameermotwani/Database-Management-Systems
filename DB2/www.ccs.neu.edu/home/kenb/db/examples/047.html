Sample Exercise

In this assignment, you must classify data from a data set that will will be provided. The data sets are available on the CCIS machines. The filename of your data set is <your name>.dat. All of the data sets have 7 columns and 100000 rows. Classify the data into 10 clusters using the k-means algorithm.

In your solution you must provide the following:

  1. The mean and standard deviation of each column, that you used for scaling the data.
  2. The SQL queries and commands that you used for finding the clusters.
  3. The source code of any programs that you used to find the clusters.
  4. The 10 cluster centers that you found.
