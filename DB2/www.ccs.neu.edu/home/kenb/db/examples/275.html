Example of a Composite Key using JPA

In this example, a Location class has a composite primary key consisting of a city and a province.

  • City.java is the persistent City class.
  • Province.java is the persistent Province class.
  • Location.java is the persistent Location class with a composite key.
  • Locater.java is the embedded class used for specifying the types of the composite key attributes in the database.
