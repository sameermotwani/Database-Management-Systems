Sample Exercise

This assignment will use the following schema: 

create table Person(
  id int primary key,
  name varchar(255) not null
);
create table Band(
  id int primary key,
  name varchar(255),
  style varchar(255)
);
create table memberOf(
  person int,
  foreign key(person) references Person(id) on update cascade on delete cascade,
  band int,
  foreign key(band) references Band(id) on update cascade on delete cascade,
  primary key(person, band)
);

Write a program that computes the number of degrees of separation between a specified musician and any other musician. For example let the specified musician be "Sean Lennon". The "Sean Lennon number" for the example data above will be:

MusicianSean Lennon Number
John Lennon1
Paul McCartney2

Linda McCartney3

Yoko Ono1
Sean Lennon0

Your program must be written in Java and use JDBC to access your database. Your program consists of a method named degreesOfSeparation that has a single parameter specifying the id of a musician. The method returns the largest degree of separation that was found. It also prints the names of the musicians and their degrees of separation, sorted first by degree of separation and then by musician name.

Note that your program must have the method specified above. A main program with a command-line argument or with interactive input may be used for testing, but you must develop the method.
