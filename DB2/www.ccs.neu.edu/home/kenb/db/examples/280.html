The If Statement in SQL

Suppose that we have a database for a company that manufactures gears. The gears are required to match specifications within a tolerance that varies from one customer to another. The following is the schema: 

create table Gear(
  id int primary key,
  accuracy double not null
)
The accuracy column is how close the gear is to its specification. The following query shows the gears and for each gear whether it is within the tolerance 0.05: 
select g.id, if(g.accuracy <= 0.05,'satisfies tolerance','does not satisfy tolerance')
  from Gear g
